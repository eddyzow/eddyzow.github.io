Who Wants To Be A Millionaire?

DEVELOPMENT: MARCH 2021

HTML

JavaScript

CSS

WebSockets

Another great quiz game I created! All controls are keyboard only so that the game can be casted with a projector in front of a real audience and controlled by an operator from the side.

Controls: Space to start a new game. Press Z twice to open a question and reveal answer choices. Press A/B/C/D to answer. Press X to reveal the correct answer, and Z twice to advance to the next question. Q/W/E for lifelines (W twice for Ask The Audience), R twice and then L to walk away. Yes, they're complicated.
