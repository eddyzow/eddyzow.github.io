How to Play Passcode

  • You'll have six tries to guess the six-digit code.
  • For each guess, if a tile turns green, the digit is correct and in the correct spot.
  • If a tile turns yellow, the digit is in the code, but not in the correct spot.
  • If a tile turns black, the digit is not in the code.
  • You'll also get the average number of digits away your answer is from the code.
