Website Archive & Evolution

What is this?

This is an archive of all my past website designs and iterations, preserving the evolution of my web development skills and personal style over the years. Each version represents a different phase in my journey as a developer, from high school through college.

Note: Version 1.0 is lost somewhere on an old computer hard drive at home!

Version 6.1 (Current)
February 2026
The current live version of my website. Clean, modern design with improved navigation and responsive layout. This is what you see when you visit the main site.
Version 6.0
August 2025
Major redesign with improved typography and cleaner layout. Added Science Olympiad section and better project showcase with dynamic loading. Enhanced mobile responsiveness.
Version 5.0
July 2025
Introduced particle effects and a more sophisticated design aesthetic. Added astronomy widgets and improved the overall visual appeal with better color schemes.
Version 4.0
June 2025
First major overhaul reflecting my engineering interests. Added engineering projects section alongside web development projects. Terminal-style typewriter effect and astronomy API integration.
Version 3.1
December 2024
Minor update to 3.0 with bug fixes and small improvements. Refined the user interface and added some missing features.
Version 3.0
June 2024
Significant redesign with modern CSS and improved project showcase. Better organization of content and enhanced visual hierarchy. This version marked my transition to more professional web design.
Version 2.0
January 2023
My high school sophomore website! Features a blog-style layout with particle effects. This was built when I was just starting to learn web development seriously. Includes my early projects like Trivia Champion and Keyboard Hero.
