Future of Online Services

3/29/23 at 6:43 PM Eastern Time (UTC-4)

Online servers will be down indefinitely while I work out problems with the system.

While I am sure that Firebase can take on real-time communication between client and server, it would involve setting up an entirely new database as well as updating the existing 17,000 lines of server code running this site.

Learning how to do this feels wasteful especially because I want to set up a MySQL database for this site, and Firebase is intended to serve as a backup while I try to find a solution and bring back the online games.

At some point, I will have to rebuild the entire backend, which will probably happen over this summer. For now, enjoy the offline games (there are a good amount of them!) I'll get live services up as soon as I can.

Switched Hosting Providers!

3/9/23 at 7:57 PM Eastern Time (UTC-5)

I've switched hosting providers! This should be able to provide you with the greatest experience on my site!

My previous host reached out to me regarding an issue with bandwidth use. They gave me 24 hours to fix the issue, which I attempted to do, but suspended my hosting after only 14. I successfully charged them back for the money!

I am now using Google Firebase. It took 10 minutes to setup and so far, has been amazing! I also moved my domain off the previous host (same company) - since Firebase is free, I can guarantee eddyzow.net will be up for the next 3 years!

Lastly, all online services are down until further notice. When back up, they will be faster than ever!
