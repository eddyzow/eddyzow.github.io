## Quick Reference

### File Locations for Common Changes

| Change | File | Line(s) |
|--------|------|---------|
| Colors | `style.css` | Top of file |
| Content | `content.js` | Entire file |
| Planets | `planets.js` | setupPlanets() |
| Galaxies | `three-scene.js` | createGalaxies() |
| Config | `config.js` | Entire file |
| HTML Structure | `index.html` | Entire file |

### Key Configuration Objects

```javascript
// In config.js
SPACE_CONFIG.colors    // All colors
SPACE_CONFIG.planets   // All planets
SPACE_CONFIG.galaxies  // All galaxies
SPACE_CONFIG.animation // All timings
```

### Keyboard Shortcut Mapping

```javascript
// In client.js setupKeyboardShortcuts()
const planetMap = {
  'h': 'home',
  'a': 'about',
  'p': 'projects',
  's': 'scioly',
  'c': 'contact'
};
```
