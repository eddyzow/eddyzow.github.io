Keyboard Hero

A rhythm game by eddyzow

Use your keyboard to be your very own rock star in this new 5-lane guitar-style rhythm game!

This isn't my first rhythm game! In early 2021, I created a game called JSBeats which was pretty much a lackluster ripoff of games like osu!mania. People really enjoyed the game, but not many actually played it.

I wasn't happy with JSBeats. It was too hard to make beatmaps; they got boring and then no one played anymore. One day, I came across a guitar arcade machine and thought it would be interesting to make something like this on the keyboard with my existing knowledge.

I stole a lot of code and resources from JSBeats in order to create Keyboard Hero (you can see the striking resemblance if you go play JSBeats, found at my website.

Special thanks to:

- Hurt Record for music/sound effects

- tetr.io, osu!mania, and RoBeats! for inspiration for almost all of my games

- Third-party libraries like socket.io, Howler.js, MongoDB, and Heroku for providing hosting

Unsplash and Pixabay for copyright-free images

- Asphyxia for being a crazy rhythm game fanatic
